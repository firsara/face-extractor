# face-extractor

extract faces as png images from files

## Installation

```sh
npm install face-extractor --save
```

## Usage

#### Doc

```sh
face-extractor input-file.doc
```

#### PDF

```sh
face-extractor input-file.pdf
```

#### Image

```sh
face-extractor input-file.png
```

### Supported File types

- doc
- docx
- odt
- pdf
- png
- jpeg
- jpg
